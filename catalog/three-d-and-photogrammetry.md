# 3D, LiDAR & Photogrammetry

Point clouds, mesh reconstructions, Gaussian splats, LiDAR collections, 3D Tiles, and photogrammetry outputs related to Ireland.

---

### Open Topographic Data Viewer
- **URL:** https://dcenr.maps.arcgis.com/apps/webappviewer/index.html?id=b7c4b0e763964070ad69bf8c1572c9f5
- **Coverage:** ROI
- **Type:** Viewer
- **Theme:** 3D / Geospatial
- **Licence:** Open (via Data.gov.ie listing)
- **Format:** Raster (processed LiDAR derivatives); linked WMS/REST services
- **Access notes:** Multi-source LiDAR mosaic from GSI, DCHG, TII, NYU; underlying service endpoints may be intermittent
- **Status:** active
- **Description:** National viewer for processed LiDAR elevation data from multiple Irish survey programmes.

### V-Sense Annotated LiDAR Point Cloud (Dublin)
- **URL:** https://v-sense.scss.tcd.ie/dublincity/
- **Coverage:** Local (Dublin)
- **Type:** Dataset
- **Theme:** 3D / Machine Learning
- **Licence:** Academic (verify at source)
- **Format:** Labelled point cloud (LAS/LAZ)
- **Access notes:** 13-class semantic segmentation derived from NYU 2015 survey; intended for ML ground truth
- **Status:** active
- **Description:** Semantically labelled 3D point cloud of Dublin city created by Trinity College's V-Sense lab, supporting machine learning and city-scale 3D reconstruction research.
