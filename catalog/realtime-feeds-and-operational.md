# Realtime Feeds & Operational Data

Frequently updated (sub-hourly or continuous) machine-readable feeds and live sensor networks.

This section cross-references entries whose primary home is in a thematic category but which also serve as notable realtime data sources.

---

### Irish Rail Realtime API
- **URL:** http://api.irishrail.ie/realtime/
- **Coverage:** ROI
- **Type:** API
- **Theme:** Transport
- **Licence:** Public-use terms; confirm reuse per endpoint docs
- **Format:** XML
- **Status:** active
- **Description:** Realtime train locations and timing data. Primary listing: [Transport & Mobility](transport-and-mobility.md).

---

### Dublin Bus GTFS-R Realtime API
- **URL:** https://developer.nationaltransport.ie/api-details#api=gtfsr&operation=gtfsr-v2
- **Coverage:** Local (Dublin)
- **Type:** API
- **Theme:** Transport
- **Format:** GTFS-Realtime (Protocol Buffers)
- **Status:** active
- **Description:** Realtime bus positions and trip updates. Primary listing: [Transport & Mobility](transport-and-mobility.md).

### TII National Roads Weather Stations
- **URL:** https://data.gov.ie/dataset/national-roads-weather-station-data
- **Coverage:** ROI
- **Type:** Dataset
- **Theme:** Transport / Environment
- **Format:** XML (DATEX II), 5-minute intervals
- **Status:** active
- **Description:** Road weather telemetry from 80+ stations. Primary listing: [Transport & Mobility](transport-and-mobility.md).
