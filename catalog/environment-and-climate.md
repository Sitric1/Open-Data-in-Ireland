# Environment & Climate

Weather forecasts and observations, hydrology, air and water quality, and marine telemetry.

---

### Met Éireann
- **URL:** https://www.met.ie/
- **Coverage:** ROI
- **Type:** API
- **Theme:** Environment / Weather
- **Licence:** Creative Commons Attribution 4.0 (via Data.gov.ie)
- **Format:** XML, JSON, CSV
- **Access notes:** Two primary APIs below; individual station CSVs also available via Data.gov.ie
- **Status:** active
- **Description:** Ireland's national meteorological service providing forecast and observation data through multiple open endpoints.

#### Met Éireann Live Text Forecast Data
- **URL:** https://data.gov.ie/dataset/met-eireann-live-text-forecast-data
- **Format:** XML API
- **Access notes:** Endpoints for national observations, weather warnings, coastal reports, farming commentary, and forecasts.

#### Met Éireann Weather Forecast API
- **URL:** https://data.gov.ie/dataset/met-eireann-weather-forecast-api
- **Format:** XML API
- **Access notes:** Query by geolocation; 1-hour intervals to 90h, 3-hour to 144h, 6-hour to 240h.

### Environment Protection Agency (EPA)
- **URL:** https://www.epa.ie/
- **Coverage:** ROI
- **Type:** Portal
- **Theme:** Environment
- **Licence:** Verify at source; some datasets on Data.gov.ie
- **Format:** GeoJSON, JSON, WMS (via GeoPortal)
- **Access notes:** GeoPortal at gis.epa.ie; EPA Maps viewer at gis.epa.ie/EPAMaps/; Air Quality Index JSON endpoint available
- **Status:** active
- **Description:** Environmental data from EPA including air quality, water quality, waste, and emissions via geoportal and API endpoints.

---

### OPW Water Levels
- **URL:** https://waterlevel.ie/
- **Coverage:** ROI
- **Type:** API
- **Theme:** Environment / Hydrology
- **Licence:** Verify at source
- **Format:** GeoJSON (API), CSV (archive downloads)
- **Access notes:** Real-Time Water Levels API at waterlevel.ie/page/api/; archive data via Hydro-Data portal
- **Status:** active
- **Description:** Realtime hydrometric monitoring data from the Office of Public Works national network of river and tidal gauges.

---

### Marine Institute Data Centre
- **URL:** https://www.marine.ie/Home/site-area/data-services/search-marine-data/access-data
- **Coverage:** ROI (marine)
- **Type:** API
- **Theme:** Environment / Marine
- **Licence:** Verify at source
- **Format:** ERDDAP (CSV, JSON, netCDF)
- **Access notes:** Irish National Tide Gauge Network API at erddap.marine.ie; multiple other marine datasets available
- **Status:** active
- **Description:** Maritime environmental data including tidal, oceanographic, and biological monitoring from Ireland's marine research agency.

### Sonitus Systems Dublin City Noise API
- **URL:** https://data.gov.ie/dataset/sonitus/resource/18befa04-7934-4ac8-b68d-5eeb7e76493d
- **Coverage:** Local (Dublin)
- **Type:** API
- **Theme:** Environment / Noise
- **Licence:** Verify at source (via Data.gov.ie)
- **Format:** JSON API
- **Access notes:** Sound level readings from Dublin City Council monitoring network; sensor locations viewable at dublincityairandnoise.ie; separate County Dublin network at dublin-noise.sonitussystems.com (not on public API)
- **Status:** active
- **Description:** Realtime sound level monitoring data from the Dublin City Council noise sensor network.
