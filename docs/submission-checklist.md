# Submission Checklist

Before submitting a new entry or update, verify the following:

## Required fields
- [ ] Name (official source name)
- [ ] URL (working link, tested today)
- [ ] Coverage (ROI / NI / All-island / Local)
- [ ] Type (Portal / API / Dataset / Map service / Viewer / Archive / Project)
- [ ] Theme (matches a catalogue category)
- [ ] Licence (stated or marked `Unclear`)
- [ ] Format (data formats available)
- [ ] Status (one of the defined status labels)
- [ ] Description (1-2 sentences, factual)

## Quality checks
- [ ] Link is reachable and returns expected content.
- [ ] Entry is not a duplicate of an existing listing.
- [ ] Entry has clear Ireland relevance.
- [ ] Licence information is sourced from the provider (not assumed).
- [ ] Entry is placed in the correct category file.
- [ ] Markdown formatting matches the entry template in CONTRIBUTING.md.

## Optional but appreciated
- [ ] Access notes (API key needed, rate limits, account required).
- [ ] Tags for discoverability.
- [ ] Note if this replaces or supersedes an existing entry.
